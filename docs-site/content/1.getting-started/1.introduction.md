# Introduction

**HoloLang** (`.hl` files) is a beginner-friendly programming language and runtime platform purpose-built for photonics engineering workloads.

## What HoloLang Does

| Subsystem | Purpose |
|---|---|
| **Language front-end** | Tokenizer → Parser → Tree-walk interpreter |
| **Holographic devices** | Laser, galvanized-mirror, sensor control |
| **Tensor/SafeTensor** | N-dimensional arrays with bounds-checking, computation graphs, and pool runtimes |
| **Mesh MDI Canvas** | Sparse 2-D tile canvas with impulse-cycle propagation |
| **Kernel VM** | Stack-based kernel, block controller, generative pipeline |
| **Network stack** | gRPC channel, WebSocket, webhook, REST API |
| **Docs & sessions** | Session lifecycle, skill registry, hierarchical doc directories |

## Design Goals

- **Readable by scientists**: syntax inspired by Rust and Kotlin but without the boilerplate
- **Domain-first keywords**: `device`, `tensor`, `mesh`, `impulse`, `emit` are first-class language constructs
- **Safe by default**: `SafeTensor` enforces min/max bounds; the kernel VM has configurable cycle limits
- **Composable pipelines**: the `->` pipe operator chains devices, tensors, channels, and APIs

## Getting Started

```bash
pip install hololang
hololang info          # show version and subsystem overview
hololang skills        # list all registered skills
hololang run examples/hello_holo.hl
hololang repl          # interactive REPL
```

Your first HoloLang program (`hello_holo.hl`):

```hololang
@session("hello")

device GreenLaser {
    type:         "solid-state-laser"
    wavelength_nm: 532
}

print("Hello, HoloLang!")
debug GreenLaser
```

Run it:

```bash
hololang run hello_holo.hl
```

Continue to [Syntax Reference](/language/syntax) to learn about all language constructs.
