# Installation

## Prerequisites

- Python **3.10+** (3.12 recommended)
- `pip` ≥ 22

## From GitHub Packages

HoloLang is published as a Python package to the GitHub Package Registry on every tagged release.

```bash
pip install hololang --index-url https://pip.pkg.github.com/somat3k/
```

## From Source

```bash
git clone https://github.com/somat3k/szalony-pomylony-pomys-owy-przemyslnik2i3-4.git
cd szalony-pomylony-pomys-owy-przemyslnik2i3-4
pip install -e ".[dev]"
```

## Optional Extras

| Extra | Packages | When to use |
|---|---|---|
| `hololang[tensor]` | `numpy>=1.26` | SafeTensor numeric ops |
| `hololang[network]` | `websockets`, `grpcio`, `aiohttp` | WebSocket / gRPC / REST |
| `hololang[all]` | all of the above | full feature set |
| `hololang[dev]` | `pytest`, `pytest-asyncio` | running tests |

```bash
# Install everything
pip install "hololang[all]"

# Developers
pip install -e ".[dev]"
```

## Docker (docs site + runtime)

A ready-to-use `docker-compose.yml` is provided at the repository root:

```bash
docker compose up
```

This starts:
- **hololang** — Python runtime container
- **docs** — NuxtJS documentation site at `http://localhost:3000`

## Verify Installation

```bash
hololang info
# HoloLang v0.1.0
# subsystems: lang · tensor · device · mesh · vm · network · docs
```
