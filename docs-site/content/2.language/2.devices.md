# Devices

HoloLang `device` declarations describe physical hardware components in a holographic system. The interpreter registers them in the runtime device registry.

## Laser

```hololang
device GreenLaser {
    type:          "solid-state-laser"
    wavelength_nm: 532
    max_power_mw:  150
    modulation:    "cw"          // "cw" | "pulsed" | "modulated"
}

device RedLaser {
    type:          "diode-laser"
    wavelength_nm: 650
    max_power_mw:   80
    modulation:    "pulsed"
}
```

## Galvanized Mirror

```hololang
device MirrorH {
    type:    "galvanized-mirror"
    axis:    "horizontal"        // "horizontal" | "vertical"
    min_deg: -30
    max_deg:  30
}
```

## Sensor

```hololang
device SensorA {
    type:           "photodiode"  // "photodiode" | "ccd" | "cmos"
    sample_rate_hz: 5000
}
```

## Supported Device Types

| `type` value | Physical component |
|---|---|
| `"solid-state-laser"` | DPSS / Nd:YAG laser |
| `"diode-laser"` | Laser diode module |
| `"galvanized-mirror"` | Galvanometer-driven mirror |
| `"photodiode"` | Single-element photodetector |
| `"ccd"` | Charge-coupled device camera |
| `"cmos"` | CMOS image sensor |

## Using Devices

After declaration, devices can be referenced anywhere:

```hololang
// Wire into a mesh tile
mesh HoloMesh(FrameBuffer) {
    tile(0, 0) -> GreenLaser
    tile(0, 2) -> MirrorH
}

// Debug device state
debug GreenLaser
```

## Beam Control

```hololang
beam {
    wavelength: 532nm
    power:      0.75          // fraction of max_power_mw
}

spectrum(450nm, 700nm)        // filter to visible range
```
