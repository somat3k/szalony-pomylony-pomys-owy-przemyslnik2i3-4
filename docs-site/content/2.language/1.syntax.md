# Syntax Reference

HoloLang files use the `.hl` extension. Syntax is inspired by Rust and Kotlin with domain-specific extensions for photonics engineering.

## Comments

```hololang
// Single-line comment
# Also a single-line comment
/* Multi-line
   block comment */
```

## Literals

```hololang
42          // integer
3.14        // float
532nm       // wavelength literal (nanometre)
"hello"     // string
true false  // booleans
null        // null value
```

## Variables

```hololang
let x = 42          // mutable variable
const PI = 3.14159  // immutable constant
```

## Operators

| Operator | Description |
|---|---|
| `+` `-` `*` `/` `%` | Arithmetic |
| `==` `!=` `<` `>` `<=` `>=` | Comparison |
| `->` | Pipe / data flow |
| `=>` | Fat-arrow pipe |
| `<-` | Left-arrow |
| `:=` | Walrus-style assignment |
| `..` | Range |
| `@` | Annotation / decorator |
| `->` in `impulse` | Impulse target |

## Functions

```hololang
fn normalize(value, min_v, max_v) {
    return (value - min_v) / (max_v - min_v)
}

let r = normalize(0.7, 0.0, 1.0)
```

## Control Flow

```hololang
if x > 0 {
    print("positive")
} else {
    print("non-positive")
}

for i in 0..10 {
    print(i)
}

while x > 0 {
    x = x - 1
}
```

## Enumerations

```hololang
enum ScanDirection {
    LEFT_RIGHT    = 0,
    RIGHT_LEFT    = 1,
    BOUSTROPHEDON = 2
}

let mode = ScanDirection.BOUSTROPHEDON
```

## Imports

```hololang
import "other_module.hl"
import "lib/utils.hl" as utils
```

## Annotations / Decorators

```hololang
@session("my-project")    // attach session context
@kernel("mesh-kernel")    // mark mesh to use a named kernel
```

## Pipe Expression

The `->` operator chains operations into a pipeline:

```hololang
invoke(HoloMesh) -> HoloGRPC -> HoloAPI
emit HoloGRPC -> "system_ready"
```
