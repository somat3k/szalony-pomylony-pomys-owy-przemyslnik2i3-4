# System Architecture

This page contains architecture and data-flow diagrams for the HoloLang runtime.
All diagrams are rendered with [Mermaid](https://mermaid.js.org/).

---

## Compilation Pipeline

A `.hl` source file passes through four stages before execution:

::mermaid
flowchart LR
    src["📄 .hl Source"] --> lex["Lexer\n(tokenize)"]
    lex --> tok["Token Stream"]
    tok --> par["Parser\n(recursive descent)"]
    par --> ast["AST\n(Program node)"]
    ast --> int["Interpreter\n(tree-walk)"]
    int --> rt["HoloRuntime\n(orchestrator)"]

    style src fill:#1f2937,stroke:#3bafc0,color:#f3f4f6
    style lex fill:#1c5f73,stroke:#3bafc0,color:#f3f4f6
    style tok fill:#1f2937,stroke:#6b7280,color:#f3f4f6
    style par fill:#1c5f73,stroke:#3bafc0,color:#f3f4f6
    style ast fill:#1f2937,stroke:#6b7280,color:#f3f4f6
    style int fill:#1c5f73,stroke:#3bafc0,color:#f3f4f6
    style rt  fill:#2094a6,stroke:#3bafc0,color:#f3f4f6
::

## AST Node Hierarchy

::mermaid
classDiagram
    class Node {
        +int line
        +int col
    }
    class Program {
        +list body
    }
    class DeviceDecl {
        +str name
        +list annotations
        +BlockStmt body
    }
    class TensorDecl {
        +str name
        +list dims
        +BlockStmt body
    }
    class MeshDecl {
        +str name
        +Node source
        +BlockStmt body
    }
    class KernelDecl {
        +str name
        +BlockStmt body
    }
    class PoolDecl {
        +str name
        +BlockStmt body
    }
    class ChannelDecl {
        +str name
        +str protocol
        +BlockStmt body
    }
    class FunctionDecl {
        +str name
        +list params
        +BlockStmt body
    }
    class ImpulseStmt {
        +Node from_tile
        +Node to_tile
        +Node payload
    }
    class PipeExpr {
        +str op
        +Node left
        +Node right
    }

    Node <|-- Program
    Node <|-- DeviceDecl
    Node <|-- TensorDecl
    Node <|-- MeshDecl
    Node <|-- KernelDecl
    Node <|-- PoolDecl
    Node <|-- ChannelDecl
    Node <|-- FunctionDecl
    Node <|-- ImpulseStmt
    Node <|-- PipeExpr
